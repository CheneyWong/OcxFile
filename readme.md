# OcxFile

## 需求
通过 javascript 操作客户端所有文件的 ocx 控件。
实现的方法有：
- create 创建文件
- read 读取指定的文件内容
  如果是文件，返回文件内容的 16 进制字符串。如果是文件夹，包含的文件和文件夹。
- write 将内容（16 进制字符串）转义为原始二进制写入文件。

##　其他

>注
网页调用时 ocx 控件如果隐藏则自定义消息失败，可通过将元素的长宽设为 0 作为替代。
